POKEBOX - La app para el seguimiento de una colección de Pokémon TCG

Podrás crear y manejar tu colección o colecciones de Pokémon TCG fácilmente, pudiendo buscar cartas específicas por set o mediante una búsqueda avanzada con filtros.
Esta aplicación utiliza una base de datos local para almacenar tu/s colección/es y visualizarlas fácilmente.
En el futuro se añadirá una forma para importar o exportar una colección, en caso de que vayas a cambiar a otro dispositivo móvil.
