POKEBOX - An app to manage your Pokemon TCG Collection
---------------------------------------

## ENGLISH

## What is Pokebox?
Pokebox is an app developed in Android Studio to manage your Pokemon TCG Collection.
You can create and delete collections, search cards by set or through filters, and check if you own the cards needed for a deck using the decklists provided by, for example, LimitlessTCG.
This app uses a local database to store your collections and view them easily.
In the future you will be able to import or export a collection to transfer it to a different device if needed.

## Why is there so much data?
Pokebox is focused on the collecting and playing aspect of the TCG, and not so much on the value of the cards.
You'll be able to see the current value of a card in the future, using an API to get that data from CardMarket.
However, this isn't a priority, and there's a chance it won't be added or updated often.

## Why does this exist?
This was done for a project. I love the TCG and I love collecting stuff, and most apps out there focus on the value aspect, or don't have enough data or search filters for my liking. This is for an assignment, yes, but also driven by passion and enjoyment of the collecting community.

### If you have any suggestions (or issues), you can open an issue on GitHub and I'll take a look at it.


## ESPAÑOL

## ¿Qué es Pokebox?
Pokebox es una aplicación desarrollada en Android Studio para manejar tu colección del JCC de Pokémon.
Puedes crear y eliminar colecciones, buscar cartas por set o mediante filtros, y comprobar si tienes las cartas necesarias para un mazo utilizando las decklists proporcionadas por, por ejemplo, LimitlessTCG.
Esta aplicación utiliza una base de datos local para almacenar tus colecciones y visualizarlas fácilmente.
En el futuro podrás importar o exportar una colección para transferirla a otro dispositivo si fuera necesario.

## ¿Por qué hay tantos datos?
Pokebox está enfocada en los lados de colección y juego del JCC, y no tanto en el aspecto de valor monetario de las cartas.
Podrás ver el valor actual de una carta en el futuro, usando una API para obtener esa información de CardMarket.
Sin embargo, esto no es una prioridad, y hay una posibilidad de que no se añada o actualice a menudo.

## ¿Por qué existe esto?
Esto ha sido hecho para un TFG. Me encanta el JCC y me eencanta coleccionar cosas, y muchas aplicaciones que existen se enfocan en el aspecto monetario, o no tienen suficientes datos o filtros de búsqueda para mi gusto. Esto es para un proyecto, sí, pero también está motivado por la pasión y disfrute que me provoca la comunidad del coleccionismo.
